# ☕ BaristaRush

Projeto desenvolvido para a disciplina de Desenvolvimento de Jogos.

## 📖 Sobre

BaristaRush é um jogo em que o jogador assume o papel de um barista. O objetivo é preparar corretamente os pedidos dos clientes antes que o tempo acabe, acumulando pontos e evitando perder vidas.
